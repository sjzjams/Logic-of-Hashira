MuscleMap这部分的实现效果.联动机制：在 onTap 触发时，通过 setState 或状态管理（如 Provider、Bloc）改变全局的日期 ID，下方的肌肉热力图组件接收到新 ID 后重新绘制各部位颜色即可。
**组件的「物质化」**。UI 元素不再是简单的出现或消失，而是像液体一样「融化」并流向下一个交互状态，保持了逻辑上的视觉连续性。
- **空间折射（Spatial Refraction）**。通过对背景材质的实时折射与反射计算，让 2D 的手机屏幕具备了类似 visionOS 的空间深度感。
- **动态路径插值**。实时计算复杂的贝塞尔曲线形变锚点，确保了不同形状组件之间切换时，每一帧的「张力」都符合物理直觉。
重点是三件事：

* ✅ 状态联动（日期 → 肌肉数据）
* ✅ 可控肌肉区域（不是图片）
* ✅ 动画过渡（颜色 + 渐进点亮）

class _MuscleMapWidgetState extends State<MuscleMapWidget>
    with SingleTickerProviderStateMixin {
  late AnimationController controller;

  @override
  void initState() {
    controller = AnimationController(
      vsync: this,
      duration: const Duration(milliseconds: 500),
    );
    super.initState();
  }

  @override
  void didChangeDependencies() {
    controller.forward(from: 0);
    super.didChangeDependencies();
  }

  @override
  Widget build(BuildContext context) {
    final state = context.watch<AppState>();

    return AnimatedBuilder(
      animation: controller,
      builder: (_, __) {
        return CustomPaint(
          size: const Size(250, 400),
          painter: MusclePainter(
            data: state.current,
            progress: controller.value,
          ),
        );
      },
    );

# 六、最关键：肌肉绘制 + 动画

## lib/ui/muscle_painter.dart

```dart
import 'package:flutter/material.dart';
import '../core/muscle_data.dart';

class MusclePainter extends CustomPainter {
  final MuscleActivation data;
  final double progress;

  MusclePainter({required this.data, required this.progress});

  Color getColor(int level) {
    final colors = [
      const Color(0xFFE0E0E0),
      const Color(0xFFFFD54F),
      const Color(0xFFAED581),
      const Color(0xFF4CAF50),
      const Color(0xFF1B5E20),
    ];

    return colors[level];
  }

  @override
  void paint(Canvas canvas, Size size) {
    final paint = Paint();

    void drawPart(Rect rect, String key) {
      final level = data.parts[key] ?? 0;

      final color = Color.lerp(
        const Color(0xFFE0E0E0),
        getColor(level),
        progress,
      )!;

      paint.color = color;

      canvas.drawRRect(
        RRect.fromRectAndRadius(rect, const Radius.circular(12)),
        paint,
      );
    }

    // 简化版身体（可换成SVG）
    drawPart(const Rect.fromLTWH(80, 60, 90, 40), "chest");
    drawPart(const Rect.fromLTWH(95, 110, 60, 60), "abs");
    drawPart(const Rect.fromLTWH(50, 60, 30, 80), "shoulder");
    drawPart(const Rect.fromLTWH(170, 60, 30, 80), "arm");
    drawPart(const Rect.fromLTWH(80, 200, 90, 60), "leg");
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => true;
}
```
